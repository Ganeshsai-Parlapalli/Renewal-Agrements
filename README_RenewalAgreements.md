# Renewal Agreements Automation System

A Python-based automation tool that monitors lease and rental agreement expiry dates, generates timely reminders, and streamlines renewal workflows to eliminate missed renewals.

## 🚀 Tech Stack

- **Language:** Python
- **Libraries:** datetime, os, schedule
- **Concepts:** OOP, File Handling, Automation

## ✨ Features

- Automatically calculates agreement due dates
- Generates renewal reminders before expiry
- Modular scripts for file handling and data parsing
- Automated alert system for upcoming renewals
- Agreement validation and record management

## 📊 Impact

- Reduced missed renewals and manual follow-up by **70%**
- Improved agreement processing efficiency by **45%**

## 🔧 Setup & Installation

```bash
git clone https://github.com/Ganeshsai-Parlapalli/Renewal-Agrements.git
cd Renewal-Agrements
pip install -r requirements.txt
python main.py
```

## 📁 Project Structure

```
Renewal-Agrements/
├── main.py
├── scheduler.py
├── validator.py
├── data/
│   └── agreements.csv
└── requirements.txt
```

## 💡 How It Works

1. Load agreement records from file
2. Calculate days remaining until expiry for each record
3. Trigger reminder alerts for agreements expiring within threshold
4. Log updated statuses and send notifications
